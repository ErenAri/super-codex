- Added benchmark scorecard diff generation and CI artifact upload for smoke and scheduled benchmark runs. (`SC2-003`)

- Made telemetry anonymous opt-in by default with explicit enable/disable controls. (`SC2-007`)

  - References: SC2-007

  - Details: Supports api-service, web-app, library, and monorepo with additive config merge behavior.

  - References: SC2-008
- Added release-train workflow for dispatch-driven canary and stable tag orchestration. (`SC2-009`)
  - Details: Release notes are now generated from structured changelog fragments before release tagging.

  - References: SC2-009
- Added post-publish npm verification for expected version and dist-tag. (`sc2-009-publish-verify`)

  - References: SC2-009
- Added growth release-gate checks with strict winner/experiment validation. (`SC2-010`)

- Added a stable MCP connector contract with official core connectors and capability discovery. (`SC2-020`)
